# Mini Prompt Engine (LangChain)

## Overview

I built a modular prompt engine using LangChain where I replaced static prompts with reusable templates. I implemented a validation layer, multiple prompt styles, and a chat-based system. The design follows a pipeline approach, making it scalable and production-ready.

---

## Features

* PromptTemplate (single & multi-input)
* Prompt variations (teaching, interview, storytelling)
* ChatPromptTemplate (role-based)
* Input validation layer
* Reusable and modular design

---

## Pipeline

User Input → Validation → Template → Prompt → Output

---

## Tech Stack

* Python
* LangChain
* Jupyter Notebook

---

```bash
pip install langchain
```
